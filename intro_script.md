## Рекомендации по съёмке

1. **Длительность**: 2–2.5 минуты (не дольше)
2. **Музыка**: фоновая, спокойная (lofi / ambient)
3. **Тон**: дружелюбный, энергичный, но без перегрузки
4. **Сцены**: чередовать fullscreen интерфейса с «говорящей головой» (опционально)
5. **Инструменты**: OBS Studio (бесплатно) для захвата экрана + любой видеоредактор (DaVinci Resolve, CapCut)
