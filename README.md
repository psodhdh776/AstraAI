# Astra AI

Когнитивный голосовой ассистент с адаптивным мышлением, эмоциональным интеллектом и графом знаний.

## Возможности

- Голосовое управление с wake-word «Астра»
- Когнитивная обработка (7-этапный конвейер: Sense → Reason → Plan → Execute → Learn → Reflect → Adapt)
- Фрактальный процессор — мультимерный анализ текста
- Система памяти — краткосрочная, долгосрочная, эпизодическая, семантическая, процедурная
- Цепочка рассуждений (Chain-of-Thought) с саморефлексией
- 24/7 фоновое прослушивание
- Домашняя автоматизация через Home Assistant REST API
- Полнотекстовый поиск по истории диалогов
- Расширяемая система плагинов
- Тема оформления и горячие клавиши

## Установка

```bash
pip install -r requirements.txt
python app.py
```

## Тестирование

```bash
pip install pytest
python -m pytest tests/
```
