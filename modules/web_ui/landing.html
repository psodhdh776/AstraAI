Архитектура

Как это работает

Модульная система из 7 уровней обработки, от восприятия до саморефлексии.

Sense

EnsembleIntentResolver — комбинирует regex, TF-IDF и контекст для точного определения интента

Reason

ReasoningEngine — 4-шаговая цепочка рассуждений с поиском в памяти

Plan

DialoguePlanner — определяет цель диалога, отслеживает дугу разговора

Execute

EngineDispatcher — weighted voting между 6 движками с учётом успешности

Learn

OnlineLearner — извлекает паттерны, стиль, факты из каждого сообщения

Reflect

Introspection — самооценка качества ответов, автоматическая коррекция

Adapt

PersonalityDynamics — эволюция черт личности, дрейф настроения, подстройка под пользователя

GitHub Actions CI/CD

3 job: test (3.12/3.13/3.14) → build (pyinstaller) → release (asset upload). Запускается на push в main и при публикации релиза.

📦

create_release.py

Скрипт для создания релиза через API: python create_release.py --tag v2.2.0 --title "..."

🚀

push_to_github.ps1

PowerShell-скрипт с авторизацией через токен из data/github_config.json. Выполняет init → add → commit → push.

🔍

REST API /github/*

GET /github/status — версия, статус git.
POST /github/push — запуск push из веб-интерфейса.

POST /github/push — запуск push.
